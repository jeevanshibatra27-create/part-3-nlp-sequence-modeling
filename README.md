# part-3-nlp-sequence-modeling
# NLP and Sequence Modeling Mini Project

## Objective
This project demonstrates text preprocessing, vectorization, baseline machine learning models, and sequence modeling using LSTM for NLP tasks.

## Tasks Covered
- Dataset understanding
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression baseline model
- LSTM sequence model
- Attention and Transformer concepts

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TensorFlow/Keras
- NLTK
